# Hayaoshi-Button-Online
Webブラウザを使った早押しボタンアプリです。URL共有するだけで複数人と早押しクイズをすることができます。

[Try it out!](https://btn.anyfrog.net)

* Express + WebSocket(Socket.io)
* Dockerでのデプロイ対応
* CircleCIでのContinuous Delivery
